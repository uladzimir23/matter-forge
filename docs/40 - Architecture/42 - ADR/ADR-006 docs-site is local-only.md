# ADR-006 docs-site is local-only

## Контекст

Исходно (в момент переноса `docs-site/` из comforthotel) предполагалось, что
matter-forge-docs будет задеплоен на Netlify — как отдельный публичный сайт
с URL типа `matter-forge-docs.netlify.app`. В volt'е было записано как
backlog-пункт.

После подумав — деплой не нужен.

## Решение

**`docs-site/` — локальная тула, не деплоится.**

- Запуск: `cd docs-site && npm run dev` → http://localhost:3010.
- Для оффлайнового статического билда: `npm run build` → `out/` → открыть
  в браузере или `npx serve out`.
- Netlify-конфига нет, CI не настраивается.

## Почему

1. **Matter-forge — приватный референс.** Vault содержит мои личные заметки
   про перенос компонентов в другие проекты (reuse log), ссылки на
   клиентские проекты, внутренние наблюдения. Не стоит публиковать это
   в открытый интернет.
2. **Один потребитель.** Volt читаю только я. Net new вклад в open-source
   нулевой. Публичный URL — оверинжиниринг.
3. **Скорость итерации.** Локальный `next dev` — HMR за миллисекунды.
   Netlify preview — ~30 секунд на билд + деплой. Для частых правок
   vault'а (session logs, catalog-entries) локально быстрее на порядок.
4. **Безопасность.** Раз нет публичного URL — redaction-policy становится
   мягче. Можно оставить ссылки на локальные пути `~/Documents/…`,
   упоминания клиентов, и прочее что в публичной доке было бы
   неприемлемо.
5. **Меньше инфры.** Не нужен Netlify-аккаунт, не нужен `netlify.toml`,
   не нужно подключать репо к Netlify, не нужно следить за билдами.

## Публичный шоукейс остаётся

Сам matter-forge (Vite React app в `src/`) деплоится на GH Pages через
Actions — см. [[ADR-005 GH Pages via Actions]]. Это публичная витрина
компонентов. Volt — приватная кухня. Разделение сознательное.

## Последствия

- **+** Меньше инфры, быстрее правки, меньше рисков с утечкой приватной
  информации.
- **+** Redaction-policy в volt'е можно ослабить (но pre-commit hook
  на секреты оставляем как страховку — см. [[.claude/skills/git-workflow]]).
- **–** Никто другой не может смотреть volt. Если когда-то захочу
  показать структуру volt'а клиенту или коллеге — сделаю экспорт вручную
  или отдам на посмотреть отдельно.
- **–** Нет auto-deploy на push → нельзя быстро проверить что volt
  рендерится на другом устройстве. Митигация: регулярно запускаем
  `npm run build` локально, это прогон всего pipeline.

## Ревизия

Если когда-то появится потребность в публичном (или защищённом по
паролю) docs-site — создаётся новый ADR «ADR-MMM docs-site goes online»
со статусом «supersedes ADR-006», настраивается деплой.
