# Metaphor Dictionary

Контролируемый словарь слов, которыми описываются компоненты, теги и
имена в matter-forge. См. скилл [[.claude/skills/design-vocabulary]] для
правил использования и расширения.

Всё по-английски (совпадает с именами `.tsx` файлов).

---

## Семьи

Слова сгруппированы по физическим/биологическим семьям. Одно слово может
использоваться кросс-категорийно (например, `resonance` — и в `acoustic`,
и в `neural`).

---

## Optical

| Слово | Значение в matter-forge |
|-------|------------------------|
| `refraction` | Преломление света через оптическую среду |
| `reflection` | Отражение, в т.ч. рекурсивное |
| `dispersion` | Хроматическое разложение на спектр (RGB-split) |
| `caustic` | Каустики — светлые пятна от преломления |

| `prismatic` | Призменное расщепление |

| `luminous` | Свечение, самосвечение |

## Technical (controlled, non-metaphoric)

Эти теги — технологии рендера, не метафоры. Один из них **обязателен** в
каждом catalog-entry.

| Слово | Значение |
|-------|---------|
| `canvas` | HTML Canvas 2D |
| `svg` | SVG с анимацией/фильтрами |
| `webgl` | WebGL / Three.js |

## State / Molecular / System

Слабее определено — это служебные семьи, метафоричность в них ниже.

### State
`state-morph`, `state-transition`, `protocol`, `entropy`, `temporal-sequence`,
`token-dynamics`

### Molecular
`atomic-assembly`, `blueprint`, `material-memory`, `material-forge`,
`fusion`, `synthesis`

### System (technical widgets, не метафоры)
`command-center`, `devops-pipeline`, `pressure-vault`, `syntax-ide`,
`accessibility-shield`, `performance-benchmark` — эти теги используются
только для компонентов из `src/components/system/`

## Расширение словаря

Добавить новое слово → апдейт этого файла в соответствующей семье →
session log с упоминанием «словарь расширен: добавили `<word>` потому
что оно встречается в `<component1>`, `<component2>`».

**Нельзя добавлять:**
- Слова-оценки (`cool`, `premium`, `fancy`)
- Слова-бренды (`apple-style`, `awwwards`)
- Слова-функции (`button`, `modal`, `card`)
- Дубли (если есть `resonance`, не добавляем `vibration`)
- Слова вне домена (matter-forge — про физику/биологию/материю/системы)

## Замена слова

Если слово неудачное и используется массово — меняем его + rewrite во
всех catalog-entries + запись в session log. Старое слово добавляем в
конец документа в секцию «Устаревшие» с указанием замены.
